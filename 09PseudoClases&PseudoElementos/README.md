Son muy utilizadas para darle efectos al DOM
<a href="https://developer.mozilla.org/en-US/docs/Web/CSS/Pseudo-classes"> - Recurso Mozilla Pseudoclases</a>

- hover
- active
- target
- focus

- Selection
- first-line
- first - letter
- before
- After
